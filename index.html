Commands

---------------------------------------------------------------------- BASICS ----------------------------------------------------------------------

!bothelp

Brings you to this page.

!leave

Makes the bot leave the current channel.

!viewers

Shows the current number of viewers.

!uptime

Shows the amount of time since the stream started.

!google <query>

Creates google link that will search for <query>

!urban <query>

Says top urban result for <query>. Limited to 140 characters.

---------------------------------------------------------------------- QUOTES ----------------------------------------------------------------------

!quote add <quote>

Adds quote.

!quote delete|remove <number ID of quote>

Removes the quote with the ID specified.

!quote get <number ID of quote>

Returns the quote with the ID specified.

!quote random

Returns random quote.

--------------------------------------------------------------------- COMMAND ---------------------------------------------------------------------

!command add <name> <response>

Adds a command with the specified name and response.

!command delete <name>

Removes the command with the specified name.

!command restrict <name> owner|mod|regular|everyone

Restricts command of specified name to the channel owner, mod, regulars or everyone.

!commands

List commands.

----------------------------------------------------------------------- REPEAT ------------------------------------------------------------------------

!repeat add <name> <delay in seconds> (amount of messages passed)

Repeats command of specified name with a delay of specified seconds if specified amount of messages have passed.

!repeat delete <name>

Removes the repeat of the specified command name.

!repeat on|off

Turn off/on repeat of the specified command name.

!repeat list

Lists the commands to repeat.
